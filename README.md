# 파이썬 구구단 프로그램 프로젝트

## 1. 개요
- AI의 제안을 받아 반복 실행 및 종료 기능이 탑재된 구구단 프로그램을 개발했습니다.
- Git Bash를 활용하여 버전 관리 및 브랜치 병합 실습을 진행했습니다.

## 2. 작업 내용
- **First Commit:** 기본 구구단 출력 기능 구현 (`main.py`)
- **Branch 생성:** 기능 개발을 위한 독립된 공간 `feature/gugudan` 생성
- **Second Commit:** 사용자가 'q'를 입력할 때까지 반복 실행되는 기능 및 예외 처리 추가
- **Merge:** `main` 브랜치로 최신 기능 병합 완료
